### Technologies

Project is created with:

- ReactJS
- JSON server
- Redux
- Thunk

### Setup

#### Server Setup

Server listens to the port `9000`

##### In terminal

```
$ cd json-server-todos
$ npm install
$ npm start
```

#### Frontend Setup

Client listens to the port `3000`

##### In terminal

```
$ npm install
$ npm start
```
